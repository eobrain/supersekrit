Colophon
========

Super Sekrit is built using the open source products below.  We thank
the contributors for their generosity.

1. [Bootstrap] [bo] as a front-end framework, including the [Glyphicons] [gi] icons
1. [Closure Compiler] [cc] for minifying JavaScript
1. [Coffeescript] [co] as a better way of writing JavaScript
1. [Glan] [gl] as a static CMS
1. [JQuery] [jq] for basic DOM hanndling
1. [JQuery-haml] [jh] for dynamic DOM creation from HAML
1. [Stanford Javascript Crypto Library] [sj] for cryptography
1. [Testem] [te] for JavaScript test-driven development

We also thank the following artists for providing their work under a
liberal Creative Commons licence:

1. Lele for ["ssh" ] [sh]
